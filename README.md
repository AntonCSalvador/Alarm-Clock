## Installation
This project requires [NodeJS](https://nodejs.org/en/) and [Git](https://git-scm.com). Also, make sure you have `build-essential` (Linux) or `xcode-select` (MacOS).

```
sudo apt install build-essential
// or...
xcode-select --install
```
